Nautilus notes

To build these docs, install mdBook from its download page (unless you have a recent Rust toolchain, in which case you would just cargo install mdbook), and run this command:

mdbook serve --open

Above command will (conveniently) open the documentation in a web browser.

See mdBook docs for more detail on how it works.

optional dependencies

Following are additional (and optional) executables to install:

  • mdbook-mermaid for displaying Mermaid diagrams (download page)
  • mdbook-toc for displaying table of contents on individual chapters (download page)
  • mdbook-admonish for displaying pretty side content (download page)
